### Docusaurus에서 비공개로 글을 작성할 수 있는 방법은 무엇이 있을까?

방법은 간단하다.

문서의 `frontmatter`에서 두 가지 옵션을 활용하면 된다.

```
---
title: 비공개로 하고 싶은 글

unlisted: true  // URL을 가진 사람만 접근이 가능하다.

draft: true // 개발환경에서만 보이고 build 시에는 보이지 않는다.
---
```

`draft: true`

- 개발 환경(npm run start)에서는 보임
- 배포 환경(npm run build)에서는 보이지 않음

![`개발 환경에서는 이미지와 같이 정보가 표시된다.`](/img/image.png)

`unlisted: true`

- URL로 직접 접근할 때만 글이 보인다.
- 블로그 목록이나 검색에는 표시되지 않는다.

![`해당 문서 접근 시 이미지와 같이 정보가 표시된다.`](/img/image-1.png)
